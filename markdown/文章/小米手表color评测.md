# 小米手表color评测：一个非专业跑者使用两个月的非专业感受


## 智能 + 手表

何为“智能手表”？
![wiki定义](./assets/wiki智能手表.jpg)
wiki的定义把手表定义为类似“可穿戴电脑”的小型设备。很显然wiki在智能手表上更关注的是“智能”二字，毕竟作为一个全新的产品类型，智能手机需要在传统手表无法企及的领域找到突破，才能把“smartwatch”里的watch真正的去掉，成为一个全新的产品类型。就如同iphone对于smartphone的突破，在未来需要一个厂商开发出真正超越“手表”定义的产品，
可目前消费市场还远没有发展到这一步，智能手表所谓的“智能”大多是手机功能的拓展：一方面是手机功能的衍生，比如监测传感器，另一方面是手机功能的简单复制。
不过这些所谓的智能，本身需要付出额外的代价：系统稳定、电池与耗电量等等。
而对于普通的消费者来说，你究竟是需要一款穿戴的智能设备？还是一款有智能功能的电子手表？

从我本身的需求角度出发，我对智能手表的关注点可能与大家不一样。

我不看重：

   1. 手机功能的重复衍生，但没有带来太多的便利
   2. 不关注soc参数
   3. 现阶段不看重所谓的智能

我看重的：

   1. 手表基本属性：外观和佩戴舒适
   2. 至少在 便利和功能 其中一个方面有其不可替代性
   3. 耗电
   4. 系统稳定：包括手表自带系统和手机app

而小米手表color，在我看来，它首先是一款手表，或者是说“电子手表”，“智能”只是附加在手表上的功能。你不会想要从这款产品中看到智能设备的发展方向，把它仅仅作为一款手表，就够了。

## 体验之前

在展开这次评测之前，先说明一下这款手表两个多月来的基本使用情况。
手表刚买来时。我的主力手机是ios系统，后来手机换成了k30 pro。而这款手表的一些核心功能在ios上是缺失的，比如：

   1. 小爱同学 卡片支付不能用
   2. 消息通知在ios上基本不可用
   3. ios上的小米穿戴app的表盘也比较少
   4. 最让人无法忍受的是与手表的连接，ios的断流特别严重。

而所有的这些问题在android上几乎都不会出现。
   
以下的体验都是基于Android手机，ios用户买这款手表要三思再三思。
这段时间，手表的固件经历了几次更新，分别增加了息屏显示和部分表盘自定义功能。未来据说还会增加游泳运动的监测。

## 手表功能
由于手表并非开放式的系统，无法安装app，只有手表本身自带的基本功能，所以把它称为大号手环也并没错。
具体的功能下面我一一为大家展开
### 心率监测

手表支持全时段不间断的心率监测，而且耗电量会有一点增加。
![小米手表color](/markdown/文章/assets/心率01.jpg)
手表的心率监测相比手环要准确稳定很多。小米手环4的心率监测非常不稳定，跑步的时候经常出现开始流汗，心率就突然掉到60 70，这也是导致我放弃使用手环的主要原因。
在使用两个个多月以来，这款手表的心率监测只出现过一次失常（同手环一样 突然掉到70左右），其余时间还是蛮稳定的。

心率监测必须贴着手腕（皮肤）才能开始，取下手表就没法进行监测
![小米手表color](/markdown/文章/assets/心率02.jpg)

这是app上心率监测图
![小米手表color](/markdown/文章/assets/心率03.jpg)

我认为心率监测是目前智能手表标配的基本功能。
心率监测的重要性有多重要？这里我用一个具体身边的例子来举例。
五一休假期间，跟朋友家庭聚餐（疫情期间外面餐馆不开业），吃饭期间，朋友突然接到一个电话，说是他的一位朋友突然猝死，直接原因就是跑步锻炼的时候突发心梗倒地。
对于像我这样一个经常跑步的人来说，心率监测和心率预警是非常非常实用的。
我跑步不追求速度，养生跑，快乐跑就行。在佩戴这款手表的两个多月时间，出现过几次心率超185的预警，很奇怪的是，在当下的情况里，身体没有任何感觉自己心率过快，速度呼吸流汗都很正常。停下来歇半分钟，心率就会恢复到160以下。
除了心率预警之外，心率也是计算运动量的重要指标。

### 抬腕亮屏 息屏显示 遮盖息屏
这三个功能需要配合使用，
手表的抬腕亮屏最好设置为灵敏，而且不如手环准确，尤其是跑步的时候。
![小米手表color](/markdown/文章/assets/抬腕亮屏01.jpg)

具体场景：
长跑过程中，如果需要看一看心率和配速，需要如下几步
1. 抬起手腕，由于小米手表的抬腕亮屏经常出现不亮屏
2. 这个时候你就需要反复放下抬起，放下抬起，
3. 眼睛也就反复一会看路一会看手腕，一来一去5秒6秒过去。
   这5秒钟你可能看着手腕就跑了二十米三十米，视线脱离路面，这实际上是非常危险的。
 
小米手环也有抬腕不良的问题，但手环是如何解决的呢：
1. 抬起手环 如果不亮 
2. 手环往下巴碰一下，屏幕就亮了，耗时不超过2秒，而且我的眼睛只需看一下屏幕即可。
   而且手环配合小米运动app可以每公里播报配速和心率。小米穿戴app每公里不但不播报配速和心率，而且会搞一个非常可笑而且不合理的东西（等会再讲。）

但是手表没有触碰亮屏，黑屏状态下如果要亮屏需要按压右侧的按键。

所以你可以看到一个滑稽的场景：

我的本意是跑步的时候根据配速和心率调整节奏，可是这款手表直接打乱我的节奏？

小米的工程师，你觉得这合理吗？

## 运动

我买手表的最主要原因是能够代替手环的运动功能，目前来看只能说小米手表color无法完全达到我的标准。
先来说说我运动的基本情况：
每天会抽空跑一小时，状态好的时候12公里，差的时候8公里，一个月大概跑20天。

以上是小米穿戴app每周会生成一个周报，大概反映一周的运动情况

手环最大的问题是心率不稳定，外观没有手表好看。
而手表的主要问题，实际上不是出在手表本身，而是小米穿戴app。
小米手表color只能连接小米穿戴app，小米手环连接的是小米运动。
从名称上来说，也能知道手表和手环定位不一样。手表是穿戴设备，手环是运动设备。
从体验上来说，也是如此，小米穿戴app的运动功能基本上满足不了运动爱好者的需求，前面也说过手表在长跑中的几个严重的体验问题。
问题：
1. 小米穿戴app无法开启运动
2. 运动过程中没有语音播报，代替的是 每公里会出现震动 和一个长达5秒钟左右的所谓奖杯的提示
   ![小米手表color](/markdown/文章/assets/奖杯01.jpg)
3. 也就是说你每跑1公里，手机震你一下，你想知道的到的是这1公里自己的运动数据，结果看到的是一个毫无意义的奖杯照片。
4. 抬腕亮屏本身不稳定，导致在运动过程中抬腕看数据是个很扰乱节奏的事情
   ![小米手表color](/markdown/文章/assets/跑步数据.jpg)
5. 无法连接Google fit，小米运动是可以连接google fit的，
6. 小米运动可以连接体重秤，小米穿戴不可以。所以你需要每次称重之后手动修改一些体重数据，虽然小米穿戴也有一些体重数据的图表，但远不如小米运动的专业和多样。

优点
1. 手表心率监测比手环准
2. 防水洗澡都没问题（照片），不管是手环还是手表我都是习惯戴着标洗热水澡，可以顺便洗一下手表上的汗渍。不过跟很多触屏设备一样，碰水触摸会失灵。
   ![小米手表color](/markdown/文章/assets/防水.jpg)
3. 外观真的不错，表盘也很丰富。手环比较不太适合一些正式场合，手表在这一方面好很多。
4. 广告少一点，最近新推出了一个运动周报的图表也还不错。

## 充电耗电量
日常使用大概每天5~7%的消耗，单次运动两小时大概消耗10%左右。
综合体验，从100%电量到16%电量 大概使用了7天，其中有5天里每天10km跑步。 而再次充满花了一小时。

## 无法忍受的问题
1. 无法触碰亮屏 + 抬腕亮屏不灵 + 运动无数据播报：
在运动过程中，你需要时不时看一下运动数据，首先app没有语音播报数据功能，如果抬腕不亮，看不到数据，我只能双手抬起，右手按左手，眼睛看着手腕，脚步还不能停，这一过程需要5到10秒钟。
实际上这三点只有做好其中一点都不会这么难受。
   1. 如果触碰能亮屏，抬腕不亮的时候，我可以用下巴碰一下屏幕
   2. 如果有语音播报，我也可以自我调整养成习惯
   3. 如果每跑1公里显示的是实际数据而不是显示长达5秒钟的奖杯图案：实在不明白我跑步过程中，你每公里震动提醒我一次就是为了看这5秒钟的奖杯图案

2. 自动亮度
   1. 手表的自动亮度传感器本身就是手环的水平。在环境亮度变化是，手表亮屏会在默认亮度反应1到2秒钟才调整到自动亮度，而且每次亮屏都这样。
      1. 具体场景：早上拿起戴着手表出门后，抬起手腕看表，手表亮屏前两秒钟是看不清的（默认亮度在日光下偏暗）
      2. 晚上抬腕，手表亮屏前两秒是很刺眼的。
   2. 而手环是如何解决这个问题的呢：手环可以设置屏幕亮度分时控制：白天6:00-18:00最大亮度 18:00-次日6:00 低亮度。
   3. 但小米手表要么手动亮度要么自动亮度。

3. 系统稳定性不佳
在使用的这一个多月时间里，出现了几次图标花屏，系统卡死。不过重启后就就好了。

## 尾声

总的来说，目前这块手表已经代替了我自己的小米手环4。在外观和功能上都能基本上满足我的日常需求。对于我来说，这款手表比一两千块左右的石英时装表要实用太多。
推荐：
1. 对手表外观有一定要求的
2. 对手表价格比较敏感的（这款手表目前价格600左右）
3. 对手表基本功能要求不高的。

不推荐哪些人群呢：
1. 女生，特别是瘦弱的女生，表盘对于女孩子来说确实大了一些
2. 跑步爱好者，在小米穿戴app的运动功能还没有大幅度改进前不推荐
3. 对手表智能功能要求太多的不推荐
   

上面说到的很多问题，实际上大部分是小米穿戴app的问题，我相信未来随着app功能的完善，这些问题都能逐步解决。
小米手表color首先是一款手表，但远远谈不上多智能，顶多是能连app的电子功能手表。
但对于很多人来说，一个不会有电量焦虑，满足基本功能，外观好看的手表是完全可以代替市面上1-2000价位的石英腕表的。小米手表color在这个标准下，确实做到了。
